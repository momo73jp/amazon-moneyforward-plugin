# Amazon × マネーフォワードME 自動メモ記入プラグイン

Amazon.co.jpの購入履歴を取得して、マネーフォワードMEの家計簿にある対応する取引のメモ欄に商品名を自動で記入するCoworkプラグインです。

## 使い方

### インストール

Coworkで以下のように入力：

/plugin install https://github.com/momo73jp/amazon-moneyforward-plugin

### 実行

手動で実行したいときはCoworkに話しかけるだけ：「Amazonのメモ更新して」

毎日自動実行したい場合は：「このプラグインを毎日朝9時に自動実行するスケジュールを設定して」

## 前提条件

- ChromeブラウザでAmazon.co.jp にログイン済みであること
- ChromeブラウザでマネーフォワードME にログイン済みであること
